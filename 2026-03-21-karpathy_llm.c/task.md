# 每日练手任务 01: 理解“超参数”

**背景**:
在我们的 V4 分析报告中，提到了 `llm.c` 是一个用于“训练”模型的项目。训练过程就像是调节一台复杂的机器，需要我们为它设定一些初始参数，这些参数控制着学习的过程，而不是在学习中产生的，我们称之为“超参数”(Hyperparameters)。

**目标受众**:
新手、非程序员、对 AI 感兴趣的学习者。

**任务**:
1.  **不需要编程**。请打开本目录下的 `source/` 文件夹，找到 `train_gpt2.c` 这个文件（你可以在 GitHub 上直接浏览）。
2.  在文件的靠前部分，找到一个名为 `// poor man's hyperparams` 的区域。
3.  **你的任务是**:
    -   用你自己的话，解释什么是“超参数”？
    -   从列表中任选一个超参数（例如 `learning_rate` 或 `batch_size`），猜想一下，如果把这个数字调得“太大”或“太小”，会对模型的训练产生什么样的影响？（例如：学习得太快/太慢？变得“死记硬背”还是“学不进去”？）

**提交方式**:
你可以在issues区，或者在你自己的笔记中写下你的理解。标准答案将于次日公布。
