# 练手任务: victorsosaMx/vsWaybar-Studio

### 任务目标
理解 Waybar 的模块化配置，并为这个项目贡献一个新的自定义模块。

### 任务步骤
1.  **环境搭建**: 在一个 Linux 虚拟机 (如 Arch Linux + Sway/Hyprland) 中安装并配置 Waybar。
2.  **应用主题**: 克隆  项目，并成功应用其中一个预设主题。
3.  **阅读文档**: 阅读 Waybar 官方关于[自定义模块](https://github.com/Alexays/Waybar/wiki/Module:-Custom)的文档。
4.  **编写新模块**: 编写一个简单的 shell 脚本，用于获取“今日一句”名言 (可以调用一个公开的 API)，并将其作为一个新的自定义模块显示在 Waybar 上。
5.  **集成样式**: 为你的新模块编写 CSS 样式，使其风格与  的某个主题相匹配。
6.  **提交贡献 (可选)**: 将你的新模块作为贡献，通过 Pull Request 的形式提交给原作者。
